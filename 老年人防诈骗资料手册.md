# 老年人防诈骗资料手册

> 基于公安部通报、央视新闻、反诈中心公告等权威来源整理
> 仅供反诈教育用途

---

## 一、保健品诈骗

### 诈骗目标
关注健康养生的老年人，尤其独居、有慢性病的老人。

### 核心套路
以"免费体检""健康讲座""专家义诊"为幌子，夸大病情、虚构疗效，推销天价保健品。

### 常见话术

| 阶段 | 话术示例 |
|------|---------|
| 邀约 | "阿姨，我们明天有一场健康知识讲座，专门请了三甲医院的专家教授，现场还有免费体检和抽奖！" |
| 建立信任 | "阿姨您就像我妈妈一样，我特别心疼您的身体……" |
| 恐吓 | "阿姨，您的检测报告出来了，这个指标很危险，不赶紧治疗会发展成癌症的！" |
| 推销 | "这款产品是诺贝尔奖技术，含有纳米级端粒酶修复因子，能逆转衰老！" |
| 促销 | "原价12800，今天活动只要3980！还送价值2000的赠品！" |
| 施压 | "名额只剩最后3个了阿姨，错过今天就没了！" |

### 诈骗步骤
1. **免费吸引**：社区发传单、送鸡蛋、送米面油，吸引老年人参加活动
2. **健康恐吓**：虚假体检，谎称发现严重疾病风险
3. **专家洗脑**：假专家讲座，用伪科学术语包装产品
4. **情感绑架**："干儿子""干女儿"式关怀，填补亲情空缺
5. **限时促销**：制造稀缺感，催促当场下单
6. **售后跟进**：持续推销更贵的产品

### 识别要点
- 真医生不会在酒店/菜场"坐诊"卖药
- 保健品不能治病，有"蓝帽子"标志才是正规保健品
- 凡宣称"包治百病""逆转衰老"的，都是骗局

## 二、投资理财诈骗

### 诈骗目标
有积蓄、担心养老金不足的老年人。

### 核心套路
以"保本高息""养老项目""原始股"为诱饵，骗取投资款后跑路。

### 常见话术

| 阶段 | 话术示例 |
|------|---------|
| 接触 | "叔叔，您知道吗？现在有一个国家扶持的养老项目，年化收益12%！" |
| 展示 | "这是政府批文，这是项目实景照片，您看多正规！" |
| 利诱 | "存10万，一年利息1万2，比银行高四倍！还送您免费入住养老公寓名额！" |
| 从众 | "您看王阿姨，上个月投了5万，这个月已经拿到分红了！" |
| 升级 | "叔叔，现在有个更好的机会——内部原始股，上市翻十倍！" |
| 收割 | "系统维护中，暂时提不了现……您再等等……"（然后失联） |

### 诈骗步骤
1. **精准筛选**：通过免费活动收集老年人信息
2. **建立信任**：豪华办公场所、伪造批文、名人合影
3. **小额返利**：初期按时付息，让老人放松警惕
4. **追加投资**：诱导老人投入更多资金甚至抵押房产
5. **卷款跑路**：资金池到一定规模后失联

### 识别要点
- 任何承诺"保本高息""稳赚不赔"的投资都是骗局
- 年化收益超过6%就要高度警惕
- 核实企业是否在银保监会/证监会备案

## 三、冒充亲友出事诈骗

### 诈骗目标
爱护子女/孙辈的老年人。

### 核心套路
冒充子女或孙辈，谎称出车祸、被抓、急病等，催促转账。

### 常见话术

| 类型 | 话术示例 |
|------|---------|
| 车祸 | "爸/妈！我出车祸了，把人撞了，对方要私了，快给我转5万块钱！" |
| 被抓 | "爸/妈，我跟人打架被抓到派出所了，要交3万保证金才能出来……" |
| 急病 | "奶奶！我急性阑尾炎要马上手术，医院要先交押金！" |
| 声音不对 | "我鼻子撞伤了/手机摔坏了/信号不好，所以声音不太一样……" |
| 保密 | "千万别告诉别人！太丢人了……" |
| 催促 | "快点啊！晚了就来不及了！" |

### 诈骗步骤
1. **号码伪装**：使用改号软件或境外号码拨打座机
2. **制造紧急**：编造车祸/被抓/急病等紧急情况
3. **声音伪装**：以"受伤""信号差"解释声音不同
4. **阻断核实**：要求保密、催促转账、不让挂电话
5. **指定账户**：要求转账到指定账户

### 识别要点
- 真子女不会只让转账不让视频核实
- 接到此类电话，先挂断，用自己的方式联系子女确认
- 任何要求"保密""别告诉别人"的紧急求助都要警惕

## 四、婚恋交友诈骗（黄昏恋杀猪盘）

### 诈骗目标
单身、丧偶、独居的老年人。

### 核心套路
通过婚恋网站/社交软件建立感情，以各种理由骗取钱财。

### 常见话术

| 阶段 | 话术示例 |
|------|---------|
| 搭讪 | "你好，看了你的资料觉得很投缘。我也一个人很多年了……" |
| 升温 | "每天跟你聊天是我最开心的事，感觉遇到了对的人。" |
| 示弱 | "今天工作好累，只有跟你说话才觉得温暖……" |
| 要钱1 | "我妈妈突然住院了，手术费还差5000……" |
| 要钱2 | "我的理财账户被冻结了，需要一笔钱解冻，解冻后马上还你。" |
| 要钱3 | "我想去看你，但是机票太贵了……" |
| 要钱4 | "我有一个内部投资机会，稳赚不赔，我们一起投吧！" |
| 拖延 | "等我忙完这阵就去看你……"（永远不去） |

### 诈骗步骤
1. **伪装身份**：假扮军官、医生、高管等体面职业
2. **情感攻势**：每日嘘寒问暖，快速建立感情
3. **制造困难**：编造家人生病、生意周转等理由借钱
4. **投资诱导**：以"为我们的未来打算"为由诱导投资
5. **消失**：钱到手后拉黑失联

### 识别要点
- 没见过面就谈钱的一律是骗局
- 要求视频/见面总是推脱的，100%有问题
- 交往过程中多次索要钱财，无论什么理由都要警惕

## 五、冒充公检法诈骗

### 诈骗目标
遵纪守法、信任权威的老年人。

### 核心套路
冒充公安、检察院、法院工作人员，以涉嫌犯罪为由恐吓转账。

### 常见话术

| 阶段 | 话术示例 |
|------|---------|
| 开场 | "你好，我是XX市公安局的，请问你是XXX吗？你的身份证号是XXXXXX对吗？" |
| 恐吓 | "你的银行卡涉嫌一起洗钱案件，涉案金额200万，你现在是犯罪嫌疑人！" |
| 施压 | "如果不配合调查，我们将对你采取强制措施，可能会判3-5年！" |
| 保密 | "这个案件还在侦查阶段，你必须保密，不能告诉任何人包括家人！" |
| 转账 | "为了证明你的清白，请将你的存款转入国家安全账户进行核查。" |
| 催促 | "请在两小时内完成操作，否则我们将冻结你所有资产！" |

### 诈骗步骤
1. **获取信息**：非法获取姓名、身份证号、住址等信息
2. **冒充身份**：来电显示伪装成110或公检法号码
3. **虚构罪名**：涉嫌洗钱、诈骗、非法出入境等
4. **发送文书**：伪造"拘捕令""通缉令"增强可信度
5. **隔离控制**：要求不告诉家人、不挂电话、去酒店开房
6. **转账验资**：要求转入"安全账户"核查

### 识别要点
- 公检法机关**绝不会**通过电话办案
- 公检法机关**绝不会**要求转账到"安全账户"
- 公检法机关**绝不会**要求你保密不告诉家人
- 直接挂断，拨打110核实

## 六、中奖/红包诈骗

### 诈骗目标
贪图小便宜、对网络不熟悉的老年人。

### 核心套路
通知中了巨额奖金，但需要先交税/手续费才能领奖。

### 常见话术

| 阶段 | 话术示例 |
|------|---------|
| 通知 | "恭喜您！您的手机号被抽中为幸运号码，获得一等奖奖金30万元！" |
| 利诱 | "这是我们公司十周年庆典活动，真实有效，有公证处公证！" |
| 要钱 | "领取奖金需要先缴纳1%的个人所得税，也就是3000元。" |
| 升级 | "恭喜！您还获得了升级大奖——一辆汽车！需要再交车辆购置税……" |
| 催促 | "兑奖有效期只剩今天了，请尽快办理！" |
| 恐吓 | "如果放弃领奖，属于违约行为，我们要追究您的法律责任！" |

### 诈骗步骤
1. **群发通知**：短信、电话、传单、刮刮卡等
2. **包装可信**：冒充知名节目组、企业庆典、公证处公证
3. **小额试探**：先让交几百元"手续费"
4. **连环套**：不断以新名目（税费、公证费、激活费）要求转账
5. **恐吓收尾**：以"违约"为由恐吓继续交钱

### 识别要点
- 没参加过的活动不可能中奖
- 真正的中奖不需要先交钱
- 任何要求"先交税后领奖"的都是诈骗
- 知名企业不会用个人账户收款

## 防骗核心原则（三不原则）

| 原则 | 说明 |
|------|------|
| **不相信** | 陌生来电自称公检法、银行、客服的，一律不信 |
| **不贪利** | 高息理财、包治百病、免费礼品，一律不碰 |
| **不转账** | 要求转账、汇款、提供验证码的，一律拒绝 |

**最关键的一条：** 遇到任何涉及钱的事情，先跟子女商量！骗子最怕的就是你跟家人商量。
